# Uniapp 外卖项目

## 项目简介
本项目是基于 **Uniapp** 的外卖平台，旨在为用户提供便捷、高效的外卖订餐服务。前端使用 **Vue** 和 **Uniapp** 构建跨平台应用，后端基于 **SpringBoot** 实现数据处理与服务支持。

## 项目特点
- **跨平台支持**: 支持 Android、iOS、小程序等多个平台。
- **用户友好界面**: 提供清晰直观的外卖订购体验。
- **高效后台服务**: 使用 SpringBoot 提供可靠的后端支持。
- **实时更新**: 实现订单状态的实时更新功能。
